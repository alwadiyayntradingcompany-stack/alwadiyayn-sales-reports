# حل سريع للمشاكل الحالية

## المشكلة الأولى: فشل الإرسال ❌

**السبب**: URL الـ Google Apps Script غير صحيح أو منتهي الصلاحية

**الحل السريع**:
1. اذهب إلى [Google Apps Script](https://script.google.com)
2. أنشئ مشروع جديد
3. انسخ الكود من ملف `google-apps-script.js`
4. غيّر المعرفات:
   ```javascript
   const SHEET_ID = 'ضع_معرف_الشيت_هنا';
   const FOLDER_ID = 'ضع_معرف_المجلد_هنا';
   ```
5. انشر كـ Web App
6. انسخ URL الجديد
7. استبدل URL في ملف `ALWADIYAYNFORM.js`

## المشكلة الثانية: التاريخ الهجري لا يظهر تلقائياً ❌

**الحل**: تم إضافة ملف `quick-fix.js` الذي يحل هذه المشكلة

**للتحقق**:
- افتح النموذج
- يجب أن يظهر التاريخ الهجري تلقائياً في الحقل

## المشكلة الثالثة: حقوق الملكية لا تظهر في الكمبيوتر ❌

**الحل**: تم تحديث CSS في ملف `ALWADIYAYNFORM.css`

## اختبار سريع:

1. **للتاريخ الهجري**: افتح النموذج وتحقق من الحقل
2. **للإرسال**: املأ النموذج واضغط إرسال
3. **لحقوق الملكية**: تحقق من أسفل يمين الشاشة

## إذا استمرت المشاكل:

1. امسح cache المتصفح
2. افتح النموذج في نافذة خاصة
3. تحقق من console للأخطاء (F12)
